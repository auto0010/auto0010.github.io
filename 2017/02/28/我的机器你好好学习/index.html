我的机器你好好学习

上次搞了 python图片转字符画 我就想要瞎搞点什么？

NLP（Natural LanguageProcessing，自然语言处理），我看到教程跑脚本，机器学习金庸，古龙的武侠小说，学习唐诗三百首。那我学习民谣歌词吧。

在自然语言处理方面，也有一个著名的深度学习模型：卡帕西模型。

斯坦福大学的安德烈•卡帕西（AndrejKarpathy）是深度学习领域的知名人物。2015年5月，他设计了一个基于递归神经网络的NLP模型，并将代码公布在 GitHub 上。

既然要学习民谣歌词，就需要抓取一些。我去网易云音乐，打开民谣歌单分类，大概有 1450 多个民谣歌单。写脚本开始抓取，去重之后，抓到了 3w 多首歌的歌词。

我要的是民谣歌词啊… 😱

这 3w 多收歌的歌词里，不仅有藏文，英文，俄文，韩语，日语。有青藏高原，有千年等一回，民谣的世界可真是丰富多彩呀。

只能过滤了，过滤之后只剩中文的歌词，大概 1w 多首歌。写到 txt 是一个 10M 多的文件。我的机器，有词可学，学富几何？学富10兆。

然后我的机器他就开始学啊学啊，我的机器配置比较低，大概12天他就能学习完成了…
