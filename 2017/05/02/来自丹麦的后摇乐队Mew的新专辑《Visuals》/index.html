来自丹麦的后摇乐队Mew的新专辑《Visuals》

没错，这篇文章就是来给大家安利，来自丹麦的后摇乐队 Mew 的新专辑《Visuals》的。

很长的一段时间，我只听了 Mew 的一首歌，就是 《Comforting Sounds》。

从1997年起至今，Mew一共发行了6张录音室专辑。真正使乐队在丹麦崭露头脚的作品是2003年发行的专辑《Frengers》，其中九分钟的《Comforting Sounds》发行后即刻被著名音乐杂志NME评为“本周最佳单曲（Single of the week）”，成为乐队进入国际市场的经典之作，无数中国乐迷也因为这首歌与Mew结缘。

我不知道是不是 newnoise.cn 把 Mew 推出来的，但是 new noise 看样子是对国内的后摇做出了很大的贡献！

关于 new noise :

New Noise创建于四川成都，由创新开放和乐于奉献的年轻人组成。

我们是一个目光开阔的演出经纪厂牌，对于真诚的音乐永远抱以激情！

我们希望把全世界的好音乐带到中国，与中国的每一位观众分享真正精彩而优异的现场演出。
