如何评价万青新歌《冀西南林路行》？

本来还准备在知乎上装个逼，写个字数比较多的回答而且还带图，结果我就嘞个去了喔，

回答建议修改：政治敏感
作者修改内容通过后，回答会重新显示恢复正常。

今天循环听了若干若干遍吧，听了这么久的万青，居然第一次听《冀西南林路行 (Live)》，一直用网易云音乐，曲库没有…

第一遍听，正如@漫画少年所说：以前没听过，旋律疏离感觉随时要跑调，随时

鸡老师的词配上董二千先生独特的石家庄风味音色，还有从头到尾都伴随的浓厚的萨克斯，听久了，还是那种久违的感觉，时代的画面感超强，就像《杀死那个石家庄人》傍晚6点下班，换掉药厂的衣裳，妻子在熬粥，我去喝几瓶啤酒。

// 或许真是的时代造就了这一切。

有一段时间一直就想特别想了解一些关于中国现代史的内容。我始终认为摇滚乐一定是具有时代的画面感，时代的情绪。比不上爱情，赞歌，违和，比不上青春，21世纪的新青年永远无法设身处地的理解八九十年代的生活给他们那一代人产生的影响，每个人都脱离不了政治，脱离不了经济。可是我了解政治已然还为时尚早，只能了解一些其他的，查阅了相关的资料有幸让我拜读了吴晓波先生的《激荡三十年》。这本书虽然写的是中国企业、商业的发展历程，书中所表述或者说记录的，可能是当时中国变革的大浪潮中，企业家如何追随时代的浪潮，以及他们睿智和从容，在改革开放的天空中自由的翱翔。但是至少描述了让我可以想象什么是1978年。作为每一个普通的人，无论你生活在哪里，生活在石家庄。即便是现在生活美好。当时呢？当时的不安，复杂，陌生，敌意，甚至愤怒，可能是没有缘由的，忽然傍晚6点下班就感觉到不安，忽然换掉了药厂的衣裳就感觉到不安，忽然妻子在熬粥就感到不安，忽然崭新万物上升幻灭如明星，我却乌云遮目感到不安。

法国思想家、1927年诺贝尔文学奖获得者伯格森曾说：“说社会的进步是由于历史某个时期的社会思想条件自然而然发生的，这简直是无稽之谈。它实际只是在这个社会已经下定决心进行实验之后才一蹴而就的。这就是说，这个社会必须要自信，或无论怎样要允许自己受到震撼，而这种震撼始终是由某个人来赋予的。

邓小平无疑就是伯格森所谓的“某个人”。

// 然而药厂上班他不会是这里的“某个人”

即便是不安过后，慢慢的也只能是，愤怒急促的流失，收回不安的目光，山河地理退入大雾后，明天是复杂的漫游。你总归还是得生活，你有妻子，你有工作，你的不安，复杂，陌生，敌意，甚至愤怒，你总归还是得生活，你有妻子，你有工作。以我的身躯为樊篱，以我的身躯为阶梯，以我的陌生和敌意其中凝聚，我去喝几瓶啤酒，在这样的生活中，他们成为时代前进的泡影，他们乌云遮目，正是有千千万万的他们，最后他们可能也没说什么。时代在进步，然后他们就消失了，如果是电影的话，结尾应该二千先生和鸡老师走在石家庄的一条叫做冀西南林路的路上，写下了这首《冀西南林路行》。
